デザインに近接を使えたらこんなことができるの？

#四大原則
#デザインルール
2024.07.30

知ってる？「近接」は関連する情報や要素を近くに配置することで、その関連性を強調し、情報のまとまりを作り出す。例えば、レストランのメニューで、同じ種類の料理を近づけて、顧客が瞬時にその料理の分類を理解できる。

デザインに整列を使ったらこんなことができるの？

#四大原則
#デザインルール
2024.07.30

知ってる？「整列」は要素の大きさや位置を揃え、決まったラインに配置することで、統一感と視覚的な調和感が生まれる。例えば、Webページでテキストや画像を左揃えにすることで、読みやすさが向上し、ページのバランスも取れる。

デザインに反復を使ったらこんなことができるの？

#四大原則
#デザインルール
2024.07.30

知ってる？「反復」は一定のルールを設け、同じデザイン要素を繰り返すことで、統一感が生まれる。例えば、Webページで同じフォントや色を使用することで、視覚的な一貫性が向上できる。また、ロゴやアイコンを統一したデザインで、ブランドの認知度が高まる。

デザインに強調を使ったらこんなことができるの？

#四大原則
#デザインルール
2024.07.30

知ってる？「強調」は大きさ、色、太さなどの差をつけることで、情報に優先度を与え、メリハリを生み出す。例えば、重要なタイトルやキーワードを大きく、目立つ色や太字で表現することで、ユーザーの視線を惹きつけることができる。

デザインに余白を使ったらこんなことができるの？

#四大原則
#デザインルール
2024.07.30

知ってる？「余白」は空白の部分であり、情報を効果的に配置し、読みやすさや視覚的なバランスを実現するために重要だ。情報のグループ化、階層化や重要な要素の強調を促進し、視覚的な混乱を減らす。また、余白をうまく使うことで、高級感や洗練された印象をを出したり、賑やかさを演出することもできる。

色が持つ心理効果とはどういうこと？

#色彩
#デザインルール
2024.07.30

知ってる？色を見た時の暖かい、冷たいと感じる温度のようなものは、色の寒暖感と呼ばれるんだ。これはいくつかの効果に関連してるんだ。例えば、実際に同じ距離にあっても、赤の方が青よりも近くにあるように見えるんだ。これは進出色と後退色と呼ばれるんだよ。それから、色の軽さや重さは明度に強く左右されるんだ。例えば、ダンボール箱の色を白などの軽い色に変えると、持ち運びの際の作業効率がアップするって言われてるんだ。

色を見るとこんなことが頭の中に浮かぶ？

#色彩
#デザインルール
2024.07.30

知ってる？特定の色を見ると、それに関係した物やイメージを思い浮かべることができる。例えば、赤と言えば、太陽や火から、熱いや情熱的などの抽象的な感情や概念までが思い起こされる。一方、青と言えば、空や海を思い出して、爽やかで澄んだイメージが伝わる。色彩心理をうまく使えば、サイトやロゴのイメージも大きく変化する。

配色の基本ルールと比率、知ってる？

#色彩
#デザインルール
2024.07.30

知ってる？サイトを作る時、配色比率が大事なことだ。一般的に「ベースカラー」（70%）、「メインカラー」（25%）、「アクセントカラー」（5%）の三つに分かれて構成されている。ベースカラーは、主に背景色などに使われるので、白・黒・グレーなどの無彩色がよく使われている。メインカラーは、サイトの印象を大きく決める色で、企業のコーポレートカラーやロゴの色から選ぶことが多い。アクセントカラーは、メリハリをつけたい時や強調したい要素がある時に使う色。

知りたい！よく使う四つの日本語フォント

#フォント
#デザインルール
2024.07.30

日本語の文字は、「明朝体」と「ゴシック体」、「筆書体」、「デザイン書体」に大まかに分かれている。明朝体は、筆で書いたように、書き始めや終わりにされている文字。ゴシック体は太さが均一で、視認性が高いとされている。筆書体は筆の印象が強い書体、デザイン書体は手書き風の書体やポップ体が含まれている。

知りたい！よく使われる四つの欧文フォント

#フォント
#デザインルール
2024.07.30

欧文語の文字も四つのカテゴリーに大別できる。一つ目は「セリフ体」と呼ばれ、Times New Romanに代表されるような縦線が太く、ウロコのある書体だ。二つ目は「サンセリフ体」と呼ばれ、線の太さが一様でウロコのない書体だ。ゴシック体なら、ArialやCalibriなどが有名だ。三つ目は「スクリプト体」で、手書きのニュアンスを表現する書体です。四つ目は「デザイン書体」で、日本語のデザイン書体と同じように、コミカルやポップなイメージを与える。

HTML、CSS、JavaScriptとはどんな関連性がある？

#HTML
#コーディング
2024.07.30

知ってる？HTMLとCSS、JavaScriptは全部Webサイトを作る時必要なプログラミング言語だけど、それぞれ違う役割を持っている。人の例にすれば、HTMLは人の骨格とで、人が必要なパーツを揃える言語だ。CSSは人の見た目やパーツの位置を調整する役割で、JavaScriptは人が動けるように働いてる。この三つはWebサイトを制作する上では必要なスキルなので、ぜひ理解してみてください！

HTMLって実際にWebデザインでこんな役割がある？

#HTML
#コーディング
2024.07.30

知ってる？HTMLはHyper Text Markup Languageの略で、Webページを作るための言語だよ。今、インターネット上にあるすべてのWebサイトで使われているんだ。 HTMLを使うと、段落や見出しを作ったり、リンクを貼ったり、画像を表示したりすることができるんだ。要するに、Webサイトの基本的な構造や機能を作るためのものなんだよ。

文字化けは怖い！これを使ったら問題を解決できる？

#HTML
#コーディング
2024.07.30

知ってる？こんな文字が出る時は「文字化け」と呼ばれる。これは、ブラウザによっては文字コードが正しく判別されなくて、文字化けの状態になってしまう。HTMLのファイルの<head>タグに<meta charset=”utf-8”>を書いたら、簡単に解消するよ。

HTMLの<h>タグってこんなことができるの？

#HTML
#コーディング
2024.07.30

知ってる？HTMLはタイトルや見出しを表現するには<h○>タグを使う。○の中には1~6までの数字を入れる、一番重要な大見出しはh1、中見出しがh2、小見出しはh3のように、違う階層にある見出しを表示できる。基本的に「h1>h6」という重要度の順に並べるイメージで考える。

リストを作りたいなら、<ol>と<ul>どっちがいい？

#HTML
#コーディング
2024.07.30

知ってる？Webページを作る時、コンテンツをリストのようにユーザーに見せたいなら、二つの方法があるよ。<ol>タグは順番に意味がある「順序リスト」を表し、一方、<ul>タグは順番が関係ない「順不同リスト」を表す。でも使う時は、どちらの場合も中に<li>タグを使ってリストの中身を書かなきゃいけないよ。

<a>タグを使ったら、別のページへ飛ばせる？

#HTML
#コーディング
2024.07.30

知ってる？自分のページから別のページやWebサイトに移動したい時は、HTMLの<a>タグを使って、その中に行き先のURLを指定すればいい。要するに、<a>タグはハイパーリンクを設置するために使う。実際に書いてみると、<a>タグで囲った部分は青色に変わり、下線が引かれてリンクになっていることがユーザーにわかるようになる。

CSSって実際にWebデザインでこんな役割がある？

#CSS
#コーディング
2024.07.30

知ってる？CSSはWebページの要素の装飾やスタイルを指示するための仕様の一つだ。簡単に言えば、これは三つのパートに分かれている。「HTMLファイルのどこ」で「何を」「どのように」するかを指定するのがCSSの役割です。 例えば、「h1」の文字色を変更したり、サイト全体の背景色を黄色にするという指示は、一般的にCSSで指定してスタイルなどをコントロールすることができる。

idとclassセレクタの違いは何って知ってる？

#CSS
#コーディング
2024.07.30

知ってる？HTMLのタグの前にidやclassの「セレクタ」を加えると、CSSで細かく指定することができるよ。でも、idとclassにはそれぞれ役割があって、使うタイミングも違う。idは同じファイル内で同じ名前を複数つけることができないので、一般的に一回しか出ない要素に使う。CSSで書く時は、id名の前に「#」をつける。一方、classは同じファイル内で同じ名前を複数つけることができるので、繰り返し使う要素に使う。CSSで書く時は、class名の前に「.」をつける。

Webページを作る時、gridとflexboxどっちを使ったほうがいい？

#CSS
#コーディング
2024.07.30

知ってる？CSSでWebサイトのレイアウトを作る時、主にグリッドとフレックスボックスという技術を使ってる。グリッドは二次元レイアウトシステムで、行（row）と列（column）の両方にアイテムを配置することができる。グリッドを使うと、複雑なページレイアウトに適している。全体のレイアウトや大きなコンテナの中の配置に便利。 フレックスボックスは一元レイアウトシステムで、アイテムを行（row）か列（column）のどちらか一方向に配置します。単純な並べ替えやアイテムの順序変更、ナビゲーションやリストなどの配置がとても便利だ。

JSって実際にWebデザインでこんな役割がある？

#JS
#コーディング
2024.07.30

知ってる？JavaScriptは、Webページに色々な動きをつけるためのプログラミング言語だよ。例えば、ボタンをクリックするとメニューが出てきたり、フォームに入力した内容をリアルタイムでチェックしたりできるんだ。Webブラウザで動くから、JavaScriptは絶対に知っておきたい技術だね。

JSにある二つの変数constとletの違いは何？

#JS
#コーディング
2024.07.30

知ってる？JavaScriptには変数というものがあるんだ。変数はデータに名前をつけることで、繰り返し利用できるようにした箱みたいなものだ。その中に、二つの種類があって、constとletがある。constは一度値を設定すると再代入できない変数で、強力な接着剤で閉じた箱を想像すると分かりやすい。一方、letは値を変更できる変数で、テープで閉じた箱のようなもの。ループや条件分岐の中で値を変更したい場合に便利だよ。

JSのobjectはデータにこんなことができるの？

#JS
#コーディング
2024.07.30

知ってる？JavaScriptのオブジェクトは、データを整理して扱いやすくするための入れ物みたいなものだよ。書き方は例えば、{name: “太郎”, age: 18}のように使う。このように、{ } の中には、左辺にプロパティ名、右辺に値を記して、オブジェクトを記述することができる。オブジェクトはたくさんのデータを一つにまとめて管理できるもので、複数の関連する情報をまとめて扱いたい時にとても便利なんだよ。

if文とelse文はプログラムに条件を分岐することができる？

#JS
#コーディング
2024.07.30

知ってる？if文は条件分岐の文章構造だ。特定の条件が満たされた時にだけ、指定された処理を実行することができる。書き方はこんな感じです。if (条件) { }。もし条件を満たせなかった場合に実行する処理があるなら、else if やelse として宣言してから新たなブロックを追加する。
